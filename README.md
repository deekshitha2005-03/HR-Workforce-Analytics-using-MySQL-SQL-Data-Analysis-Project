# 📊 HR Workforce Analytics using MySQL

## 📌 Project Overview

This project demonstrates an end-to-end SQL-based HR Workforce Analytics solution using MySQL. The goal is to analyze employee data and generate meaningful business insights related to employee attrition, salary, performance, experience, and job satisfaction.

Instead of directly creating dashboards, this project focuses on strengthening SQL skills by following a real-world analytics workflow.

---

## 🎯 Objectives

- Design and create an HR database
- Import and validate employee data
- Perform data cleaning
- Conduct Exploratory Data Analysis (EDA)
- Solve real business problems using SQL
- Apply advanced SQL concepts including:
  - CASE Statements
  - Subqueries
  - Correlated Subqueries
  - Common Table Expressions (CTEs)
  - Window Functions
  - Views

---

## 🛠️ Tools Used

- MySQL 8.0
- MySQL Workbench
- SQL
